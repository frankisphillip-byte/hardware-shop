# Hardware Shop

A hardware store management system built with React, TypeScript, and Supabase.

## Features
- Inventory Management
- Sales/Point of Sale
- Customer Management
- Supplier Management
- Product Categories
- Low Stock Alerts
- Reports & Analytics

## Deployment
- Frontend: https://hardware-shop-five.vercel.app
- Backend API: Vercel Serverless
- Database: Supabase
